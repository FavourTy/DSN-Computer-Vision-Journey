# 🚀 My Computer Vision Journey

Welcome to my personal repository documenting my path into the fascinating world of Computer Vision! This repository serves as a centralized hub for my notes, projects, and experiments as I progress from foundational mathematics to advanced deep learning models.

## 🌟 Background

My journey began with building a strong foundation in Mathematics for Machine Learning via **Coursera**, equipping me with the linear algebra, calculus, and probability necessary to understand what goes on under the hood of machine learning algorithms.

Currently, I am actively expanding my skill set by participating in **Data Science Nigeria (DSN) Deeptech Cohort 3**, where I am diving deep into Computer Vision and practical image processing techniques.

## 📂 Repository Structure

This repository is organized into several key areas, reflecting the progression of my learning:

- **[`/image_preprocessing_technique`](./image_preprocessing_technique)**: Contains Jupyter notebooks and scripts focused on fundamental image manipulation, feature extraction, and traditional computer vision algorithms.
- **[`/DL` (Deep Learning) `./DL`](./DL)**: Dedicated to deep learning experiments, including neural network architectures, model training, and evaluations using modern frameworks.
- **[`/image_processing_app`](./image_processing_app)**: An interactive web application built with Streamlit. It allows users to upload images and apply various preprocessing techniques (like translation, rotation, scaling, blurring, and edge detection) in real-time, providing an intuitive way to visualize how these algorithms affect image data.
